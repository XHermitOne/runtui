# rad/codegen.py - Code Generator

Generate Python code from widget schemas.

---

## Functions

### `generate_python`
```python
def generate_python(data: dict) -> str
```

Generate a complete runnable Python file from a RAD design dict.

**Parameters:**
- `data` (`dict`): The RAD design dictionary containing app config, windows, widgets, and menus

**Returns:** `str` - Complete Python code

### `save_python`
```python
def save_python(code: str, path: str) -> None
```

Write generated Python code to a file.

**Parameters:**
- `code` (`str`): The Python code
- `path` (`str`): File path to write

## Features

- Auto-generates unique widget IDs (lbl1, btn1, etc.)
- Handles radio button groups
- Generates menu bar code
- Creates event handler stubs
- Uses AbsoluteLayout for widget positioning
- Includes shebang for direct execution

## Usage

```python
from runtui.rad.deserializer import load_json
from runtui.rad.codegen import generate_python, save_python

# Load design
data = load_json("my_app.json")

# Generate code
code = generate_python(data)

# Save to file
save_python(code, "my_app.py")
```
