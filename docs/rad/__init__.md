# rad Package

Rapid Application Development tools for runtui.

## Modules

| Module | Description |
|--------|-------------|
| `schema` | Widget schema definitions |
| `serializer` | Serialize widgets to JSON |
| `deserializer` | Deserialize JSON to widgets |
| `codegen` | Generate Python code from schema |
| `runner` | Run RAD-designed applications |
| `design_surface` | Visual design surface |
| `menu_editor` | Menu structure editor |

## Overview

The RAD package provides tools for visually designing runtui applications:
- **Schema**: JSON-based widget definitions
- **Serializer/Deserializer**: Save and load designs
- **Codegen**: Generate Python code from designs
- **Design Surface**: Visual widget placement
- **Menu Editor**: Edit menu structures
