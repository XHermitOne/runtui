# core Package

Core types, events, and utilities for the runtui framework.

## Modules

| Module | Description |
|--------|-------------|
| `types` | Fundamental types: Point, Size, Rect, Color, Attrs, Cell, BorderStyle |
| `keys` | Key constants, modifiers, and mouse button/action enums |
| `event` | Event base classes, typed events, and dispatch system |
| `unicode` | Unicode character width utilities |
| `timer` | Timer handle for scheduled callbacks |
| `event_loop` | Async event loop for the application |
| `pty_process` | PTY process management for terminal widgets |
| `key_encode` | Key encoding utilities |

## Exports

The core package exports fundamental types and utilities used throughout the framework.
