# dialogs Package

Modal dialogs for runtui applications.

## Available Dialogs

| Dialog | Description |
|--------|-------------|
| `Dialog` | Base modal dialog class |
| `MessageBox` | Simple message display |
| `OpenFileDialog` | File open dialog |
| `SaveFileDialog` | File save dialog |
| `FormDialog` | Form input dialog |

## Usage

Dialogs are modal and block until dismissed. They center themselves on screen when shown.
