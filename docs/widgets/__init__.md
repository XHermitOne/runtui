# widgets Package

Widget library for runtui.

## Available Widgets

| Widget | Description |
|--------|-------------|
| `Widget` | Base class for all widgets |
| `Container` | Widget that holds and lays out children |
| `Button` | Clickable button |
| `Calendar` | Date picker calendar |
| `Checkbox` | Checkbox with label |
| `ColorPicker` | Color selection widget |
| `ContextMenu` | Right-click context menu |
| `DropDownList` | Dropdown selection |
| `ImageWidget` | Image display |
| `Label` | Static text display |
| `ListBox` | Scrollable list of items |
| `MenuBar` / `Menu` / `MenuItem` | Menu system |
| `PasswordInput` | Password entry field |
| `PipeTerminalWidget` | Terminal with pipe input |
| `ProgressBar` | Progress indicator |
| `RadioButton` / `RadioGroup` | Radio button selection |
| `StaticImage` | Static Image display |
| `TerminalWidget` | Embedded terminal |
| `TextArea` | Multi-line text editor |
| `TextInput` | Single-line text input |
| `VScrollbar` / `HScrollbar` | Scroll bars |
