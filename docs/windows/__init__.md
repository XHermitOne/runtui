# windows Package

Window management for runtui desktop applications.

## Modules

| Module | Description |
|--------|-------------|
| `window` | Window widget with title bar and controls |
| `window_manager` | Z-ordering, focus, tiling, and cascading |
| `taskbar` | Desktop taskbar widget |

## Overview

The windows package provides a desktop-like window management system:
- **Window**: Top-level windows with title bar, minimize/maximize/close buttons
- **WindowManager**: Manages multiple windows with z-ordering and focus
- **Taskbar**: Bottom bar showing open windows
